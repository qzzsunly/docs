9. System Requirements

Spring Boot 2.0.0.BUILD-SNAPSHOT requires Java 8 and Spring Framework 5.0.1.BUILD-SNAPSHOT or above. Explicit build support is provided for Maven (3.2+), and Gradle 4.

9.1 Servlet containers

The following embedded servlet containers are supported out of the box:

NameServlet Version

Tomcat 8.5

3.1

Jetty 9.4

3.1

Undertow 1.3

3.1

You can also deploy Spring Boot applications to any Servlet 3.0+ compatible container.
